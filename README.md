🎓 Learnova – AI Study Assistant

📌 Overview

Learnova is an AI-powered study assistant designed to help university students organize their academic tasks, manage study materials, and receive instant academic support through an AI chatbot.

This project was developed as part of a Human-Computer Interaction (HCI) course, focusing on user-centered design, usability, and AI-assisted learning.

⸻

✨ Features

📅 Daily Study Planner
✅ Task Management (Add, Edit, Delete)
🤖 AI Chatbot
📚 Study Materials Library
📎 File Upload Support
🔔 Smart Study Organization
